# 🧠 Lumina – Student Trust Core

### Privacy-First Digital Identity & Security Infrastructure for Campuses

Lumina is a privacy-focused student trust and digital identity platform designed to secure campus digital ecosystems while protecting student data. It provides a unified system that follows students from onboarding to academic and social activities, ensuring safe interactions without exposing personal information.

The platform introduces a new model of campus security where identity verification, phishing protection, and academic analytics are performed using privacy-preserving technologies. Lumina enables institutions to gain meaningful insights without accessing sensitive student data, creating a transparent and trusted digital environment.

---

# 🚀 Overview

Lumina Student Trust Core is built as a secure and scalable infrastructure that integrates:

* Privacy-first digital identity for students
* Browser-level phishing detection and protection
* Gamified trust badge system for cybersecurity awareness
* Zero-data identity verification for clubs and labs
* Anonymized academic analytics using differential privacy

The system ensures that students maintain control over their digital identity while institutions receive only aggregated insights needed to improve academic outcomes.

---

# 💡 Key Features

### 🔐 Trust Badge System

A gamified digital trust scoring mechanism that evaluates cybersecurity awareness and promotes safe online behavior among students.

### 🌐 Browser Protection Layer

Real-time phishing detection through a browser extension that provides explainable warnings and interactive learning before accessing suspicious links.

### 🆔 Secure Lumina Digital ID

RSA-based encrypted identity system with JWT verification, enabling secure access to campus services without revealing personal information.

### 🏫 Zero-Data Club & Lab Verification

Token-based identity verification and QR check-ins allow students to join clubs or labs without sharing phone numbers or emails.

### 🧪 Privacy-Preserving Academic Analytics

Aggregated analytics using differential privacy help institutions identify learning gaps without tracking individual student performance.

# 🏗 System Architecture

Lumina follows a modular and scalable architecture:

* Student dashboard and admin dashboard interfaces
* Browser extension for phishing detection
* Secure backend for authentication and analytics
* Token-based identity verification system
* Privacy layer ensuring anonymized data processing

The system is designed to evolve from a prototype to a full institutional deployment with scalable infrastructure.
